Сведения о системе

Клавиатурный тренажер

Клавиатурный тренажёр – программа, предназначенная для обучения набору на компьютерной клавиатуре. Основными целями тренажера являются:

  • научить слепому методу печати;
  • научить использовать для набора все десять пальцев рук;
  • увеличить скорость набора;
  • уменьшить количество опечаток;
  • улучшить ритмичность набора (что позволяет уменьшить усталость при наборе).

Вход в систему

Если у вас еще нет аккаунта, вам необходимо пройти регистрацию в системе. Для это выберете пункт "Регистрация" в форме входа, введите логин, дважды введите пароль.

Если у вас уже есть аккуаунт, вам необходимо пройти авторизацию в системе. Для это выберете пункт "Вход" в форме входа, введите логин и пароль.

Тестовое задание

Тестовое задание поможет определить ваш уровень подготовки на текущий момент. обращайтесь к нему в любой момент вашего обучения для отслежтвания прогресса. Для выполнения тестового задания, перейдите в пункт меню "Тестовое задание"

Уровни сложности и упражнения

Для выбора упражнения, перейдите в пункт меню "Упражнения". В таблице расположены все существующие упражнения для данного уровня сложности.

Чтобы выбрать упражнение другого уровня сложности, нажмите на кнопку с указанным уровнем сложности и выбиретенеобходимый уровень сложности в выпадающем списке.

Тренажер

Выбрав упражнение, вы сможете приступить к выполнению задания. Для начала упражнения нажмите на пробел. Выше поля ввода вы сможете следить за результатами выполнения упражнения: количество символов, скорость набора, ошибки, время и счет.
